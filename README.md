# Etsy US Market Seasonal Listing Calendar

An interactive, responsive single-page web application designed for Etsy sellers targeting the United States market. This tool helps sellers apply the "Golden Rule" of Etsy SEO by scheduling product listings at least **2 to 3 months** before peak demand.

## 🚀 Key Features

* **47 Curated US Holidays & Niche Seasons**: Covers major shopping periods (Christmas, Halloween, Mother's Day), professional appreciation days (Nurses, Teachers, Doctors, IT), and micro-occasions (National Pet Day, Grandparents Day).
* **Automatic Timeline Calculator**: Generates the **Ideal Listing Start (3 Months Prior)** and the **Hard Deadline (2 Months Prior)** for each event, automatically adjusting for dynamic dates (e.g. Easter, Mother's Day, Thanksgiving) across target years (2026, 2027, and 2028).
* **"Prepare Right Now" Dashboard**: Automatically reads your computer's date to cluster actions into three prioritized phases:
  * **Start Listing Now (3mo prior)**: Key seasons opening their optimal listing window today.
  * **Last Call (2mo prior)**: Urgent windows to capture search traffic before it's too late.
  * **Upcoming (45 days prep)**: Future events to begin designing products and tags for.
* **Custom Event Scheduler**: Allows you to add custom niche events or local celebrations to compute custom listing windows.
* **Search & Filters**: Instantly query by tags, holiday names, and status filters (All, Active, Last Call, Upcoming).

## 🛠️ Technology Stack

* Built completely using **HTML5, CSS3, and modern Vanilla JavaScript**.
* **Zero dependencies**: No installation required. Runs entirely client-side in any modern browser.
